**Object Counting with YOLO26 by Ultralytics**
This repository contains a simple implementation of real-time object counting using YOLO26 by Ultralytics. Using the official code from their documentation, I tested it on a video with zero fine-tuning and achieved 60%+ accuracy on the first try — the results were surprisingly great straight out of the box. The script supports both video file and webcam input, and can be further improved with custom fine-tuning for specific use cases.

Reference: https://docs.ultralytics.com/guides/object-counting/#what-is-object-counting
